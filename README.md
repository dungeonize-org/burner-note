# Burner Note
Burn and keep. Close things without deleting them.

## Japanese / 日本語

```
Burner Note は、「物理的にノートを買い換える行為」をデジタルの世界に持ち込むためのアイデアおよび実装である。

多くの人は、メモ帳やノートを使い切ると買い換える。
ページが埋まり、役目が終わるからだ。
その結果、文房具売り場に足を運び、短い単位で新しいノートを選ぶ。

しかし PC やスマートフォンの世界では事情が異なる。
Apple、Google、Microsoft などが提供する標準の Note アプリを、
一度使い始めると、それを何年も使い続ける。

ノートは終わらない。
内容は積み重なり続ける。
区切りを付けるタイミングが存在しない。

Burner Note は、この違和感から生まれた。

Concept

Burner Note の基本思想は次の通り。

作る。
使う。
終わらせる。
残す。

ノートは消すものではない。
役目が終わったら閉じるものである。

Burner Note は、使い捨てでありながら、消されないノートである。

Why Burner

Burner とは、隠すための言葉ではない。
一時的に使い、役目が終わったら手放せるという意味である。

Burner Note は次のことを許可する。

途中でやめてもいい。
完成させなくてもいい。
終わらせてもいい。

ただし、削除しなくていい。

What a Note Is

Burner Note におけるノートは「表現」ではなく「コンテナ」である。

ノートは次のものを理解しない。

数式の意味。
手書きの内容。
図の構造。

ノートが行うのは、保存、表示、並べ替え、終了（Close）のみである。

One Note = One File

Burner Note は、1ノートを 1ファイル（または 1パッケージ）として扱う。

ノートには次が含まれる。

名前。
表紙。
本文（主にテキスト）。
添付物。
作成日時。
状態（open / closed）。

Close されたノートは削除されない。
棚に戻されるだけである。

Pages

ノート作成時にページサイズを固定する（A4、B5 など）。

編集時は連続スクロール。
ページ分割は表示や印刷の段階で行う。

これは思考を止めず、物理ノートの感覚を残すためである。

History

Undo / Redo は線形ではなく木構造を想定する。

書き直した過去は消されない。
選ばれなかった分岐も残る。

Cloud

クラウド同期は必須ではない。

Burner Note はローカル所有を前提とする。
Export / Import を最優先する。

クラウドは必要な場合のみ補助的に使われる。

Beyond Notes

この考え方は Note に限定されない。

一時的な Messenger。
期間限定のチャット。
短期プロジェクト。
日記。
感情の吐き出し。
下書き用ツール。

共通点は一つ。
ずっと開いておく必要がないこと。

Burner Note は「終わらせる前提で作られたコンテナ」の総称である。

What This Is Not

Burner Note は次を目指さない。

多機能化。
大量ユーザー。
生産性競争。

目指すのは、
手放せない人が少数いるツールである。
```

## English

```
Burner Note brings the act of replacing physical notebooks into the digital world.

People replace paper notebooks when they are full.
Pages end. Roles finish. A new notebook begins.

Digital note apps behave differently.
Default note apps on PCs and smartphones are used endlessly.
Notes never end.
Content keeps accumulating.
There is no natural closure.

Burner Note starts from this discomfort.

Create.
Use.
Close.
Keep.

Notes are not meant to be deleted.
They are meant to be closed.

Burner does not mean hiding.
It means temporary by design.

Burner Note allows you to stop, leave unfinished, and close things without deleting them.

A note is a container.
It does not interpret content.
It stores, displays, organizes, and closes.

Each note is a single file or package.
Closed notes are kept, not removed.

Page size is fixed at creation.
Editing is continuous.
Pagination happens only at view or print time.

Undo and redo form a tree.
Alternative histories are preserved.

Cloud sync is optional.
Local ownership comes first.

This idea extends beyond notes.
Temporary messengers, session chats, short-term projects, journals, drafts.

Burner Note is a family of containers designed to be closed.

It does not aim for mass adoption or feature competition.
It aims to be a tool a few people cannot let go of.
```
